概述

Chrome 开发者工具

Chrome 开发者工具是一套内置于Google Chrome中的Web开发和调试工具，可用来对网站进行迭代、调试和分析。

提示: 寻找最新版本的 Chrome 开发者工具, Chrome Canary 总是有最新的 DevTools。

打开 Chrome 开发者工具

  • 在 Chrome 菜单中选择 更多工具 > 开发者工具
  • 在页面元素上右键点击，选择 “检查”
  • 使用 快捷键 Ctrl+Shift+I (Windows) 或 Cmd+Opt+I (Mac)
